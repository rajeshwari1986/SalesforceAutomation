Build and dependency management tool --> Project management tool

POM.xml --> Project object model

Writing code
Compiling code
Testing code
Packaging (.war, .jar)
Deploying code

Build lifecycle:
Default --> Project build and deployment
Clean -->  Cleans the artifacts generated by previous build
site --> project site documentation


Lifecycle is made of phases: 

Validate --> all the information, files  : resouseces:resources
Compile --> compile source code : compiler:compile
test --> unit test for the source code (frame) -->  resources:testresources, sureFire:test
integration-test --> compiler:testCompile
Package --> creating the package .jar, .war --> jar:jar
Verify --> Quality criteria --> verify:verify
install --> Install the package local/ repository  --> install:install
deploy --> Web server  --> deploy:deploy

Each phase will have a goal, Goal can be acheived by using a plugin
